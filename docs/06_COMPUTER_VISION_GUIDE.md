# Computer Vision Guide

- Applies starting with the first sprint that introduces inference code
- All inference through Detector abstraction
- Standard output schema: class, confidence, bbox
- Confidence threshold configurable
- Benchmark mAP50, mAP50-95, Precision, Recall
- Separate train/val/test datasets
- Never hardcode model paths
- Support image, video, RTSP in their respective sprints only
- Future-ready for ByteTrack and ONNX/TensorRT
- This guide does not authorize inference code during Sprint 0
